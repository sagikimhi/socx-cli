SoCX is a command-line toolbox for EDA development teams.

SoCX can also be used in your own python project/library/script and provides an
extensive API with many useful features targeted at EDA development.

## Installation

Requirements: PythonÂ 3.12 or newer.

### Install with `pipx`

```bash
pip install socx-cli
```

### Install with `uv`

```bash
uv tool install socx-cli
```

Upgrade to the latest release at any time with `pip install --upgrade` or
`uv tool install --upgrade`.

## Verify the Installation

```bash
socx --help
socx version
```

If you see the top-level help and the current version, the CLI is ready to use.
